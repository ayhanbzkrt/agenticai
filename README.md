# AI'Han Ajan Simülatör

**AI'Han Ajan Simülatör**, yapay zekâ ajanlarının nasıl çalıştığını görselleştiren interaktif bir web portalıdır.  
Bu proje; bir AI ajanının **planlama**, **hafıza**, **araç kullanımı**, **algı** ve **kendi kendini geliştirme** süreçlerini sade, modern ve etkileşimli bir arayüz üzerinden deneyimlemeyi amaçlar.

**AI'Han Agent Simulator** is an interactive web portal designed to visualize how AI agents work.  
The project demonstrates the core capabilities of an AI agent — **planning**, **memory**, **tool use**, **perception**, and **self-improvement** — through a clean, modern, and interactive interface.

---

## Özellikler / Features

- Yapay zekâ ajanı mantığını görselleştiren interaktif simülasyon  
- Hazır senaryolar ile hızlı deneyim  
- Özel görev girerek ajan akışını başlatma  
- 5 temel ajan yeteneğinin ayrı ayrı sunumu  
- Adım adım işlem akışı ekranı  
- Hafıza paneli, olay günlüğü ve istatistik kartları  
- Modern, koyu tema ve responsive tasarım  
- Tek dosyada çalışan yapı (`HTML + CSS + JavaScript`)  

- Interactive simulation that visualizes AI agent logic  
- Ready-made scenarios for quick exploration  
- Custom task input for launching personalized agent workflows  
- Dedicated presentation of 5 core agent capabilities  
- Step-by-step execution flow screen  
- Memory panel, event log, and statistics cards  
- Modern dark theme with responsive design  
- Single-file architecture (`HTML + CSS + JavaScript`)

## GitHub Pages ile Yayınlama / Deploy with GitHub Pages

### TR
1. Yeni bir GitHub deposu oluşturun  
2. `index.html` dosyanızı yükleyin  
3. Repo ayarlarına girin  
4. **Pages** bölümünü açın  
5. Branch olarak `main` seçin  
6. Root klasörünü kullanın  
7. Kaydedin ve yayın linkinizi açın  

### ENG
1. Create a new GitHub repository  
2. Upload your `index.html` file  
3. Go to repository settings  
4. Open the **Pages** section  
5. Select the `main` branch  
6. Use the root folder  
7. Save and open your published site link
