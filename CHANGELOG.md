- AI'Han Ajan Simülatör ana arayüzü eklendi

- Yapay zekâ ajanı yetenek rozetleri eklendi
- Hazır senaryo kartları oluşturuldu
- Özel görev girme alanı eklendi
- Adım adım simülasyon akışı tasarlandı
- Hafıza paneli, olay günlüğü ve istatistik kutuları eklendi
- İlerleme göstergesi ve ajan durum alanı oluşturuldu
- Responsive koyu tema arayüz geliştirildi
- Tek dosyada çalışan HTML, CSS ve JavaScript yapısı kuruldu
- Footer alanına LinkedIn, GitHub ve Başa Dön bağlantıları eklendi

### Included Scenarios
- Seyahat Planlama

- Araştırma Görevi
- Yaratıcı Proje

### Documentation
